## Context

**Client:** Developer services company, ~50 employees, ~1,400 repos, ~300 GitLab groups, multi-model AI tooling, mostly VS Code. They know we're Claude Code power users — this is one of the reasons they want to work with us.

**Two products from one company:**
- **PLUR Enterprise** = Organizational Learning (subscription) — this is the main offer
- **Datacore Enterprise** = AI Development Team (future, Q3/Q4) — Chief of Staff, Insight Agent, Onboarding Companion

**What we're selling:**
1. **PLUR Enterprise subscription** — EUR 70/seat list, EUR 49/seat Founding Partner (30% off). 50 seats, 12-month commitment starting at go-live.
2. **Knowledge Engineering** — one-time custom project. Scan 1,400 repos, extract conventions, build custom ingest pipeline. Start with 5 repos, feedback loop, iterate, scale. Standard consulting rate, no discount.
3. **Datacore Enterprise** — AI roles priced monthly. Scoped together after PLUR is in place. Vision section, indicative pricing only.

**Key framing:**
- "Organizational Learning" not "Shared Memory" — the system gets smarter, not just stores things
- Founding Partner (not design partner) — guaranteed best rate, roadmap influence
- No charges before go-live — integration + test run included
- They pay for custom work on THEIR codebase, not for building our platform
- Phase 3 AI roles are outcomes, not infrastructure — Chief of Staff answers questions, Insight Agent surfaces what you didn't ask, Onboarding Companion guides new hires

**Pricing defense (EUR 49/seat Founding Partner, EUR 70 list):**
- GitHub Copilot Enterprise: $39-60/seat — no persistent memory, no learning
- Augment Code Standard: $60/seat — project memory but no correction-based learning
- Sourcegraph Cody Enterprise: $59/seat — code search, no persistent memory
- Zero competitors have correction-based learning + org knowledge sharing
- a16z April 2026: "user corrections and feedback loops are the future of AI learning"
- Saving 15 min/day per dev = 5x ROI, break-even in 2.5 months

**Timeline:**
- Early May: contract
- May W1-2: integration + first 5 repos
- May W2-3: feedback loop with tech leads
- May W3-4: test run + next batch
- June: go-live, 50 users, subscription starts
- June-Aug: Knowledge Engineering completes
- Q3/Q4: Datacore Enterprise scoped

**Negotiation notes (internal):**
- 30% is the opening. Budget for 35-40% as acceptable outcome if pushed.
- Don't lead with 40% — let them negotiate there.
- "No charges before go-live" is a concession card — use it when they push on commitment length.
- Knowledge Engineering has no discount — it's custom work at standard rate.
- Founding Partner = guaranteed best rate (MFN-equivalent without the jargon).
